又、純正機能を損なう事はありません。

メーカーナビゲーション連動の場合
外部入力 ( VTR ) で地上デジタルチューナーを接続した場合、
チューナー付属のリモコンを使用せずに車両ステアリングスイッチの SEEK スイッチで
選局が出来ます。( 2ボタンの登録が可能 )
市販ナビゲーション+地デジチューナーの場合
市販ナビゲーション連動の場合
市販のナビゲーション操作機能と地上デジタルチューナーの選局を
市販のナビゲーション機能やオーディオ操作をステアリングスイッチで音量調整や
ステアリングスイッチ純正機能そのままに操作出来ます。( 8ボタンの登録が可能 )
選曲等純正機能そのままに操作出来ます。( 8ボタンの登録が可能 )
